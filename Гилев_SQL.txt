Написать SQL запрос по отчету 2, который выведет клиентов с регионом выдачи онлайн и выданные в 2024 годуSELECT ClientID FROM table2WHERE SubdivisionName = ‘Онлайн’   AND EXTRACT(YEAR FROM TO_TIMESTAMP(issueDateTimestamp, 'DD.MM.YYYY HH24:MI')) = 2024;2.  Написать SQL запрос по отчету 2, который выведет три поля: статус, количество клиентов в статусе, сумму выданных займов в статусеSELECT Status, COUNT(ClientID) AS client_count, SUM(LoanAmount) AS loan_sumFROM table2GROUP BY Status;3. Написать SQL запрос, который выведет все поля отчета 1 и отчета 2SELECT  tb2.NumContract, tb2.ContractID, … -- перечислить все поля из таблицы 2,tb1.«номер контакта», … -- Перечислить все поля из таблицы 1  FROM table2 tb2FULL OUTER JOIN table1 tb1ON tb2.NumContract = tb1.«номер контакта»* В третьем запросе Необходимо заменить «номер контакта» на название соответствующей колонки из таблицы в БД* Предполагаем, что table2 – таблица, в которой хранится отчет 2 в БД; table1 – таблица, в которой хранится отчет 1 в БД* Не используем выбор всех полей через *, а перечисляем их названия для повышения производительности и читаемости запроса
